# Dotfiles

These are my configuration files (dotfiles) to customize my development environment
with LazyVim, Zellij, LazyGit, and Fish.
These files contain settings and configurations
that I've found useful for my daily workflow.

## Contents

- **LazyVim**: Configuration for the Vim text editor.
- **Zellij**: Configuration for the terminal multiplexer Zellij.
- **LazyGit**: Configuration for the Git user interface, LazyGit.
- **Fish**: Configuration for the Fish shell.

## Installation

- Clone this repository to your local machine

```bash
git clone https://github.com/farbautie/dotfiles
```

- Copy the files to your home directory

```bash
cp -r dotfiles ~/.config
```

## Customization

Feel free to customize these files according to your own preferences and needs.
You can modify colors, keyboard shortcuts, plugins, and other configurations as you wish.
Remember to keep a backup of your previous configurations before making significant changes.
